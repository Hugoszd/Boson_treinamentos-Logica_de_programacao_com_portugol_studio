# Objetivo
Projeto teste escrevendo quantos números o usuário determinar 

<details>
<summary>Código</summary>

```c
programa
{
	inteiro num, quant_num
	
	funcao inicio()
	{
		escreva("Digite quantos números quer: ")
		leia(quant_num)

		para (num = 1; num <= quant_num; num++){
			escreva("Número: ", num, "\n")
		}
	}
}
```
